# BIOTICA Generated Plots

This directory contains automatically generated plots and figures.

## Plot Types
- `ibr_distribution_*.png` - IBR distribution histograms
- `time_series_*.png` - Time series analysis
- `correlation_*.png` - Correlation matrices
- `alert_map_*.png` - Geographic alert maps
- `trend_*.png` - Trend analysis plots

## Generation
Plots are generated daily by the reporting system using:
- matplotlib (Python)
- ggplot2 (R)
- plotly (interactive)
